My Work

Various group and individual projects I have worked on.

Granny's Recipe Box

Granny's Recipe Box is an app that allows a user to upload recipes using photos and screenshots where the text is read from the photo using an OCR service then stored in the recipe as text. Users are able to then edit the recipe, see other user's recipes, and leave notes on recipes for themselves or other users. The goal of this app is to allow users to easily save and edit recipes they find online or in cook books then share them with other users. Granny's Recipe Box is built using JavaScript, React.js, HTML, CSS, and React Bootstrap.

Nutshell (Group Project)

Nutshell is a single page CRUD social media web app created by four team members utilizing React.js, vanilla JavaScript, HTML, CSS, and React Bootstrap. It allows users to add events, tasks, news articles, friends, and send messages to each other. The goal of Nutshell is to allow users to connect and keep track of each other easily.

Tabloid (Group Project)

Tabloid is a single page CRUD blog web app created by four team members utilizing SQL, C# and ASP.NET using MVC. It has two different types of users, An Administrator or Author. Administrators can add new Tags, or Categories for blogs, activate or deactivate users, and change user types. Authors are able to add new posts, see other posts, and leave comments.
